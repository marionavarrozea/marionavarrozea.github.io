# Mario Navarro Zea — Personal Portfolio

A responsive, static personal portfolio website built to showcase my academic background, professional experience, and technical competencies
as an Aerospace Engineer specializing in space systems, propulsion, and AIT.

## About

This repository contains the source code for my personal portfolio. The site is designed to be lightweight, fast, and accessible, 
without relying on heavy front-end frameworks. It provides a clean user interface to navigate my engineering journey—from my foundational studies 
at the Universidad de Sevilla and Politecnico di Milano to my current work at the European Space Agency (ESA).

## Technical Features

- **Semantic HTML5:** Clean, accessible markup structure.
- **Modern CSS3:** Utilizes CSS variables for consistent theming, Flexbox/Grid for responsive layouts, and a fluid grid system that automatically scales to any device size.
- **Vanilla JavaScript:**
  - **Intersection Observer:** Triggers smooth reveal animations as sections scroll into view.
  - **Telemetry Scroll Bar:** A custom scroll progress bar that tracks viewport position in real-time.
  - **Dynamic Navigation:** A sticky navbar that shrinks on scroll and a mobile-friendly hamburger menu.
- **Analytics Integration:** Fully integrated with Google Tag Manager (GTM) and Google Analytics 4 (GA4) for traffic telemetry.

## Tech Stack

* **Frontend:** HTML5, CSS3
* **Scripting:** Vanilla JavaScript (ES6+)
* **Analytics:** Google Tag Manager / GA4
* **Deployment:** [Live on GitHub Pages](https://marionavarrozea.github.io)
